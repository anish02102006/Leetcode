The problem **“Binary Tree Postorder Traversal”** on LeetCode asks you to return the **postorder traversal** of a binary tree’s nodes’ values. Postorder traversal visits nodes in the order: **Left → Right → Root**.

Here’s a complete explanation of all approaches with **Java code**:

---

## **1. Recursive Approach**

### **Idea**

* Recursively traverse the left subtree
* Recursively traverse the right subtree
* Visit the root

### **Code (Java)**

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */

import java.util.*;

class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        postorder(root, result);
        return result;
    }
    
    private void postorder(TreeNode node, List<Integer> result) {
        if (node == null) return;
        postorder(node.left, result);
        postorder(node.right, result);
        result.add(node.val);
    }
}
```

**Time Complexity:** O(n)
**Space Complexity:** O(h) for recursion stack (h = tree height)

---

## **2. Iterative Approach using Stack (Two Stacks)**

### **Idea**

* Use one stack to traverse nodes
* Push root → left child → right child
* Use a second stack to reverse the order to get postorder

### **Code (Java)**

```java
import java.util.*;

class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        if (root == null) return result;

        Stack<TreeNode> stack1 = new Stack<>();
        Stack<TreeNode> stack2 = new Stack<>();

        stack1.push(root);
        while (!stack1.isEmpty()) {
            TreeNode node = stack1.pop();
            stack2.push(node);
            if (node.left != null) stack1.push(node.left);
            if (node.right != null) stack1.push(node.right);
        }

        while (!stack2.isEmpty()) {
            result.add(stack2.pop().val);
        }

        return result;
    }
}
```

**Time Complexity:** O(n)
**Space Complexity:** O(n)

---

## **3. Iterative Approach using Single Stack**

### **Idea**

* Use a single stack and a pointer to track last visited node
* Traverse left subtree fully, then right, then root

### **Code (Java)**

```java
import java.util.*;

class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        Stack<TreeNode> stack = new Stack<>();
        TreeNode current = root;
        TreeNode lastVisited = null;

        while (!stack.isEmpty() || current != null) {
            if (current != null) {
                stack.push(current);
                current = current.left;
            } else {
                TreeNode peekNode = stack.peek();
                if (peekNode.right != null && lastVisited != peekNode.right) {
                    current = peekNode.right;
                } else {
                    result.add(peekNode.val);
                    lastVisited = stack.pop();
                }
            }
        }

        return result;
    }
}
```

**Time Complexity:** O(n)
**Space Complexity:** O(h) (stack stores at most tree height)

---

## **4. Morris Postorder Traversal (O(1) Space)**

* Advanced method without stack or recursion
* Reverse the traversal of rightmost path of left subtree

*(Typically used in competitive programming for O(1) space)*

---

### **Summary of Approaches**

| Approach                 | Time Complexity | Space Complexity |
| ------------------------ | --------------- | ---------------- |
| Recursive                | O(n)            | O(h)             |
| Iterative (Two Stacks)   | O(n)            | O(n)             |
| Iterative (Single Stack) | O(n)            | O(h)             |
| Morris Traversal         | O(n)            | O(1)             |
