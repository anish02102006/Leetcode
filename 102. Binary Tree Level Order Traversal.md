The problem **“Binary Tree Level Order Traversal”** asks to return the nodes of a binary tree **level by level**, from top to bottom. Let’s go through all approaches with **Java code**.

---

## **1. Recursive Approach (DFS with Level Tracking)**

### **Idea**

* Use depth-first search
* Pass current **level** as argument
* Add node values to a list corresponding to their level

### **Code (Java)**

```java
import java.util.*;

class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        dfs(root, 0, result);
        return result;
    }
    
    private void dfs(TreeNode node, int level, List<List<Integer>> result) {
        if (node == null) return;
        if (result.size() == level) {
            result.add(new ArrayList<>());
        }
        result.get(level).add(node.val);
        dfs(node.left, level + 1, result);
        dfs(node.right, level + 1, result);
    }
}
```

**Time Complexity:** O(n)
**Space Complexity:** O(h) (recursion stack, h = height of tree)

---

## **2. Iterative Approach using Queue (BFS)**

### **Idea**

* Use a queue to traverse the tree **level by level**
* For each level, process all nodes in the queue

### **Code (Java)**

```java
import java.util.*;

class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        if (root == null) return result;

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);

        while (!queue.isEmpty()) {
            int size = queue.size();
            List<Integer> level = new ArrayList<>();

            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                level.add(node.val);
                if (node.left != null) queue.offer(node.left);
                if (node.right != null) queue.offer(node.right);
            }

            result.add(level);
        }

        return result;
    }
}
```

**Time Complexity:** O(n)
**Space Complexity:** O(n) (queue stores at most one level of nodes)

---

## **3. Iterative Approach using Two Queues**

### **Idea**

* Use two queues to alternate between current level and next level
* Less commonly used since single queue is simpler

### **Code (Java)**

```java
import java.util.*;

class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        if (root == null) return result;

        Queue<TreeNode> current = new LinkedList<>();
        Queue<TreeNode> next = new LinkedList<>();
        current.offer(root);

        List<Integer> level = new ArrayList<>();
        while (!current.isEmpty()) {
            TreeNode node = current.poll();
            level.add(node.val);
            if (node.left != null) next.offer(node.left);
            if (node.right != null) next.offer(node.right);

            if (current.isEmpty()) {
                result.add(level);
                level = new ArrayList<>();
                Queue<TreeNode> temp = current;
                current = next;
                next = temp;
            }
        }

        return result;
    }
}
```

**Time Complexity:** O(n)
**Space Complexity:** O(n)

## **4. Level Order Traversal Using Recursion + Queue (Hybrid)**

* Sometimes recursion is combined with a queue for tracking
* Similar in concept to DFS with level tracking

---

## **5. Summary of Approaches**

| Approach                       | Time Complexity | Space Complexity | Notes                                  |
| ------------------------------ | --------------- | ---------------- | -------------------------------------- |
| DFS Recursive (level tracking) | O(n)            | O(h)             | Elegant for recursion lovers           |
| BFS Queue (single queue)       | O(n)            | O(n)             | Most commonly used                     |
| BFS Two Queues                 | O(n)            | O(n)             | Less common, explicit level separation |
