# Intuition
The problem asks us to flatten a binary tree into a linked list following preorder traversal (root → left → right).

We can solve this in two ways:

Extra space approach (easy to implement): Perform preorder traversal, collect nodes in a list, and then rebuild the linked list.
In-place approach (optimized): Rearrange pointers during traversal without using extra space.
# Approach
# Approach 1: Using Extra Space (Preorder Traversal + Reconstruction)
Do a preorder traversal and store all nodes in a list.
Iterate through the list and connect each node’s right to the next node, while setting left = null.
Approach 2: In-Place (Optimized Morris-like Traversal)
Start from the root node.

If the current node has a left child:

Find the rightmost node of the left subtree.
# Attach that node’s right to the current node’s right subtree.
Move the left subtree to the right and set left = null.
Move to the right child and repeat.

# Complexity
Approach 1 (Extra Space):

Time Complexity: O(n)
Space Complexity: O(n)
Approach 2 (In-Place):

Time Complexity: O(n)
Space Complexity: O(1)

# Code
// ✅ Approach 1: Extra Space (Preorder Traversal)
class Solution {

    public void flatten(TreeNode root) {
        if (root == null) return;
        List<TreeNode> nodes = new ArrayList<>();
        preorder(root, nodes);

        for (int i = 0; i < nodes.size() - 1; i++) {
            nodes.get(i).left = null;
            nodes.get(i).right = nodes.get(i + 1);
        }
    }

    private void preorder(TreeNode node, List<TreeNode> nodes) {
        if (node == null) return;
        nodes.add(node);
        preorder(node.left, nodes);
        preorder(node.right, nodes);
    }
}

// ✅ Approach 2: In-Place Optimized
class Solution {

    public void flatten(TreeNode root) {
        TreeNode current = root;
        while (current != null) {
            if (current.left != null) {
                TreeNode temp = current.left;
                while (temp.right != null) {
                    temp = temp.right;
                }
                temp.right = current.right;
                current.right = current.left;
                current.left = null;
            }
            current = current.right;
        }
    }
}
